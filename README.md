# zip-crack

## como funsiona?
esta herramienta realiza un ataque de diccionario si el ataque de diccionario falla pasa a un ataque de fuerza bruta 

## como usar
para iniciar, esta herramienta en Dragon-Black primero deve instalarla en **Dragon-Black**

### instalacion
para instalar esta herramienta en **Dragon-Black** ejecute el siguiente comando dentro de **Dragon-Black**
```bash 
plugin install zip-crack
```
la herramienta se instalarla porque lla esta disponible en el repo de plugins de Dragon-Black 

## como usar 
una vez instalada dentro del entoro de **Dragon-Black**
simplemente ejecute el siguiente comando dentro de
